# Algorithmic Backtester

A lightweight, modular Python framework for backtesting trading strategies against historical data. This tool allows you to evaluate performance metrics like CAGR, Sharpe Ratio, and Maximum Drawdown before risking capital.

## 🚀 Features
* **Vectorized Backtesting:** High-performance testing using NumPy and Pandas.
* **Performance Analytics:** Automated calculation of risk-adjusted returns.
* **Visualization:** Equity curve and drawdown charts.
* **Modular Design:** Easily swap out data sources or strategy logic.
